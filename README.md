# 🚦 Traffic Vehicle Detection System

AI-based traffic surveillance system using YOLOv8.

## 🔥 Features
- Detect cars and bikes
- Green bounding boxes
- Real-time vehicle counting
- 2x speed video processing
- Clean UI (bottom-left counter)

## 🛠️ Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)

## 📁 Project Structure
```
traffic-vehicle-detection/
│── input/
│── output/
│── models/
│── main.py
│── requirements.txt
```

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```

## 🎥 Output
